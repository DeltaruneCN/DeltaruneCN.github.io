文字教程


  1. 在官网下载游戏客户端,游戏键盘和配置文件(UT/DR配置文件不相同)
  2. 依次点击设置-导入配置-选择DeltaruneGKP-fixed.gkp
  3. 点击自定义按键-显示/隐藏键盘-按音量+键
  4. 勾选设置内的,左右滑动动作
  5. 点击进入宏设置
    按宏1 输入
    上|19,5,1
    按宏2 输入
    下|20,5,1
    按宏3 输入
    左|21,5,1
    按宏4 输入
    右|22,5,1
  6. 回到主页面,点击第一个按钮-启用游戏键盘
  7. 点击第二个按钮-点击游戏键盘
  8. 进入游戏,按音量+按键会出现手柄,疯狂左右滑动会出现键盘
  9. 按←→键会出现上下左右的箭头,选中头部然后再按←→,打开大写锁定键,按Z选中(记住,每一次回到键盘都要重新大写)
  10. 依次选择躯干,腿,取名,进入游戏,按Z继续对话,按X跳过对话
  11. 一直像这样玩到Kris和Susie跌入地下世界
  12. 打开大写锁定,按C键打开菜单,按→键直到选中最右边的选择框
  13. 按Z选中,按↓键选中控制(Control),再按,Z,键
  14. 找到选中(Confirm)并按Z键,然后字的颜色会变成红色
  15. 按两下左下角的手柄图标,回到手柄,按A键
  16. 按手柄下键,A键,B键,设置取消键
  17. 用同上方法设置菜单键
  18. 选择完成(Finish)按A,按两下B键即可
  19. 完成以上步骤即可用游戏键盘正常游玩
